Support — Daily Work Log

You can use any of the support options below. If you don't have a support email or domain yet, Option A works with just a GitHub account.

Option A: GitHub Issues (recommended)

Create (or use) a GitHub repository for support and enable Issues. Then replace the placeholder link below with your real Issues URL.

Placeholder Issues URL: https://github.com/YOUR-USERNAME/daily-work-log-support/issues/new/choose

After your repo exists, edit this page and change the link. App stores accept this page as a valid Support URL.

Option B: Temporary Google Form

  1. Create a free Google Form with fields: Name, Email, Message.
  2. Set “Anyone with the link can respond”.
  3. Paste the share link here.

Option C: Email (later)

When you're ready, create a support inbox and add a mailto: link here.
